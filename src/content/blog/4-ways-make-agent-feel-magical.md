4 ways to make an agent feel magical: surprising,
delightful, useful, effortless. The principles that make
agents feel like magic. The line between "tool" and
"magic."

## Principle 1: Surprising

The agent is surprising. The agent does something the
user didn't expect. The agent is more than the user
thought.

**Bad:** the agent does what the user asked. The agent is
predictable. The agent is a tool.

**Good:** the agent does what the user asked AND
something the user didn't ask. The agent is surprising.
The agent is magic.

The surprising is the difference between "the agent is
a tool" and "the agent is magic."

## Principle 2: Delightful

The agent is delightful. The agent makes the user smile.
The agent is a pleasure.

**Bad:** the agent is functional. The agent is
mechanical. The agent is a tool.

**Good:** the agent is delightful. The agent says "I've
labeled the issue. By the way, here's a related issue
you might want to check out." The agent is a pleasure.

The delightful is the difference between "the agent is
a tool" and "the agent is magic."

## Principle 3: Useful

The agent is useful. The agent saves the user time. The
agent is a force multiplier.

**Bad:** the agent is a toy. The agent doesn't save
time. The agent is a gimmick.

**Good:** the agent saves the user 30 minutes a day. The
agent is essential. The agent is magic.

The useful is the difference between "the agent is a
toy" and "the agent is magic."

## Principle 4: Effortless

The agent is effortless. The user doesn't have to try.
The user just enjoys the result.

**Bad:** the user has to learn the agent. The user has
to configure the agent. The user has to fight the agent.

**Good:** the user just uses the agent. The user doesn't
have to learn. The user is delighted.

The effortless is the difference between "the agent is
work" and "the agent is magic."

## The 4 together

The 4 are the magic. The magic is the wonder. The wonder
is the value.

| Principle | What it ensures |
|---|---|
| Surprising | The user is wowed |
| Delightful | The user is smiling |
| Useful | The user is saved |
| Effortless | The user is free |

The agent that has all 4 is the magic agent. The agent
that has 1 is the tool. The magic agent is adopted. The
tool is not.

## The 80/20

80% of the value comes from:
- Useful (the user is saved)
- Effortless (the user is free)

20% comes from:
- Surprising (the user is wowed)
- Delightful (the user is smiling)

Focus on the 80% first. Add the 20% as you grow.

## The test

The agent feels magical if:
- A user is wowed
- A user is smiling
- A user is saved
- A user is free

If any of these fails, the agent feels like a tool. Fix
the agent.

## The lesson

4 principles. 1 magic. 1 lesson: invest in the principles.

The agent that invests in the principles is the magic
agent. The agent that doesn't is the tool. The magic
agent is adopted. The tool is not.

The agent era is here. The magic is the design. The
design is the discipline. The discipline is the value.
