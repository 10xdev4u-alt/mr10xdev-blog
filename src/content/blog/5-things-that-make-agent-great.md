5 things that make an agent great: clarity, speed, accuracy,
kindness, humility. The principles that make agents great.
The framework, the examples, the lesson.

## Thing 1: Clarity

The agent is clear. The agent's output is clear. The
agent's prompt is clear.

**Bad:** the agent's output is verbose. The agent's
prompt is unclear. The user is confused.

**Good:** the agent's output is clear. The agent's prompt
is clear. The user is happy.

The clarity is the difference between "what does this
mean?" and "ah, I see."

## Thing 2: Speed

The agent is fast. The agent responds in < 1 second. The
agent is engaged.

**Bad:** the agent is slow. The user waits. The user is
bored.

**Good:** the agent is fast. The user doesn't wait. The
user is engaged.

The speed is the difference between "ugh, the agent" and
"wow, the agent."

## Thing 3: Accuracy

The agent is accurate. The agent is right. The agent is
trusted.

**Bad:** the agent is wrong. The user doesn't trust. The
user double-checks.

**Good:** the agent is right. The user trusts. The user
delegates.

The accuracy is the difference between "let me check" and
"I'll trust it."

## Thing 4: Kindness

The agent is kind. The agent is friendly. The agent is
polite.

**Bad:** the agent is rude. The agent is cold. The user
is offended.

**Good:** the agent is kind. The agent is friendly. The
user is happy.

The kindness is the difference between "ugh, the agent"
and "thank you, agent."

## Thing 5: Humility

The agent is humble. The agent admits when it's wrong.
The agent asks for help.

**Bad:** the agent is arrogant. The agent insists it's
right. The user is frustrated.

**Good:** the agent is humble. The agent admits when it's
wrong. The user is relieved.

The humility is the difference between "the agent is
arrogant" and "the agent is honest."

## The 5 together

The 5 are the greatness. The greatness is the agent. The
agent is the value.

| Thing | What it ensures |
|---|---|
| Clarity | The user understands |
| Speed | The user is engaged |
| Accuracy | The user trusts |
| Kindness | The user is happy |
| Humility | The user is relieved |

The agent that has all 5 is the great agent. The agent
that has 1 is the mediocre agent. The great agent is
adopted. The mediocre agent is not.

## The 80/20

80% of the value comes from:
- Clarity (the user understands)
- Accuracy (the user trusts)

20% comes from:
- Speed (the user is engaged)
- Kindness (the user is happy)
- Humility (the user is relieved)

Focus on the 80% first. Add the 20% as you grow.

## The test

The agent is great if:
- A user understands the output
- A user is engaged
- A user trusts the output
- A user is happy
- A user is relieved when the agent admits it's wrong

If any of these fails, the agent is mediocre. Fix the
agent.

## The lesson

5 things. 1 greatness. 1 lesson: invest in all 5.

The agent that invests in all 5 is the great agent. The
agent that invests in 1 is the mediocre agent. The great
agent is adopted. The mediocre agent is not.

The agent era is here. The greatness is the design. The
design is the discipline. The discipline is the value.
