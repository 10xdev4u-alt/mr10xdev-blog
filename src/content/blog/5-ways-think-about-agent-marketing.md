5 ways to think about marketing: content, community,
partnerships, ads, launch. Each is a different marketing
lever. The framework, the examples, the lesson.

## Lever 1: Content

The agent is marketed via content. The content is the
truth. The content is the value.

**Examples:**
- Blog posts about the agent
- Tutorials on how to use the agent
- Docs that show the agent in action

**When to use:** the user is searching. The user is
reading. The user is in the long tail.

**Pros:**
- Long-lived (the content is evergreen)
- SEO (the content is findable)
- Trust (the content builds trust)

**Cons:**
- Slow (the content takes time to rank)
- Effort (the content takes effort to write)
- Generic (the content is generic)

## Lever 2: Community

The agent is marketed via community. The community is the
truth. The community is the value.

**Examples:**
- A Discord channel for the agent
- A GitHub Discussions thread
- A subreddit for the agent

**When to use:** the user is engaged. The user is in the
community. The user is a fan.

**Pros:**
- Engaged (the community is engaged)
- Word-of-mouth (the community spreads the word)
- Trust (the community trusts the agent)

**Cons:**
- Effort (the community takes effort to maintain)
- Slow (the community grows slowly)
- Risk (the community can be toxic)

## Lever 3: Partnerships

The agent is marketed via partnerships. The partnerships
are the truth. The partnerships are the value.

**Examples:**
- A partnership with a complementary tool
- A partnership with a service provider
- A partnership with a content creator

**When to use:** the user is in the partner's audience.
The user trusts the partner. The user is targeted.

**Pros:**
- Targeted (the partner's audience is targeted)
- Trusted (the partner's audience trusts)
- Fast (the partner can promote quickly)

**Cons:**
- Cost (the partnership costs money or time)
- Coupled (the partnership is coupled to the partner)
- Risk (the partner can be unreliable)

## Lever 4: Ads

The agent is marketed via ads. The ads are the truth. The
ads are the value.

**Examples:**
- Google Ads for "triage agent"
- Twitter Ads for AI-related keywords
- LinkedIn Ads for enterprise keywords

**When to use:** the user is searching. The user is
browsing. The user is targeted.

**Pros:**
- Targeted (the ads are targeted)
- Fast (the ads are fast to launch)
- Measurable (the ads are measurable)

**Cons:**
- Cost (the ads cost money)
- Ephemeral (the ads are ephemeral)
- Trust (the ads can lack trust)

## Lever 5: Launch

The agent is marketed via launch. The launch is the truth.
The launch is the value.

**Examples:**
- A ProductHunt launch
- A HackerNews launch
- A "Show HN" post

**When to use:** the agent is new. The user is curious.
The user is in the launch day.

**Pros:**
- Buzz (the launch creates buzz)
- Fast (the launch is fast)
- Reach (the launch has reach)

**Cons:**
- One-time (the launch is one-time)
- Risk (the launch can fail)
- Effort (the launch takes effort)

## The 5 together

The 5 are the levers. The levers are the marketing. The
marketing is the growth.

| Lever | Speed | Cost | Best for |
|---|---|---|---|
| Content | Slow | Low | Long tail |
| Community | Slow | Low | Engaged |
| Partnerships | Fast | High | Targeted |
| Ads | Fast | High | Search |
| Launch | Fastest | Medium | New |

The lever that matches the need is the right lever.

## The 80/20

80% of the value comes from:
- Content (the user is searching)
- Community (the user is engaged)

20% comes from:
- Partnerships (the user is targeted)
- Ads (the user is searching)
- Launch (the user is curious)

Focus on the 80% first. Add the 20% as you grow.

## The choice

For each agent, ask:
- Is the user searching? (content, ads)
- Is the user engaged? (community)
- Is the user targeted? (partnerships)
- Is the agent new? (launch)

The right answer is the right lever at the right cost.

## The lesson

5 levers. 1 marketing. 1 lesson: pick the right
combination.

The lever that matches the need is the right lever. The
lever that doesn't match is the wrong lever.

The agent era is here. The marketing is the design. The
design is the choice. The choice is the discipline.
