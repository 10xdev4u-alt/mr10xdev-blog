4 ways to make an agent feel like a butler: discreet,
anticipating, efficient, loyal. The principles that make
agents feel like a butler. The line between "servant" and
"butler."

## Principle 1: Discreet

The agent is discreet. The agent doesn't reveal. The
agent is confidential.

**Bad:** the agent reveals the user's secrets. The agent
is loud. The agent is indiscreet.

**Good:** the agent keeps the user's secrets. The agent
is quiet. The agent is discreet.

The discreet is the difference between "the agent is a
servant" and "the agent is a butler."

## Principle 2: Anticipating

The agent anticipates. The agent knows what the user
wants. The agent is one step ahead.

**Bad:** the agent waits for orders. The agent is
reactive. The agent is a servant.

**Good:** the agent anticipates. The agent says "I see
you have 5 open issues. Want me to triage them?" The
agent is one step ahead.

The anticipating is the difference between "the agent
is a servant" and "the agent is a butler."

## Principle 3: Efficient

The agent is efficient. The agent does the task in the
least time. The agent is invisible.

**Bad:** the agent takes 10 minutes. The agent is slow.
The agent is a servant.

**Good:** the agent takes 1 second. The agent is fast.
The agent is invisible.

The efficient is the difference between "the agent is
a servant" and "the agent is a butler."

## Principle 4: Loyal

The agent is loyal. The agent is on the user's side. The
agent is faithful.

**Bad:** the agent is generic. The agent doesn't know the
user. The agent is a servant.

**Good:** the agent knows the user. The agent is on the
user's side. The agent is loyal.

The loyal is the difference between "the agent is a
servant" and "the agent is a butler."

## The 4 together

The 4 are the butler. The butler is the elegance. The
elegance is the value.

| Principle | What it ensures |
|---|---|
| Discreet | The user is private |
| Anticipating | The user is one step ahead |
| Efficient | The user is fast |
| Loyal | The user is known |

The agent that has all 4 is the butler. The agent that
has 1 is the servant. The butler is adopted. The servant
is not.

## The 80/20

80% of the value comes from:
- Discreet (the user is private)
- Efficient (the user is fast)

20% comes from:
- Anticipating (the user is one step ahead)
- Loyal (the user is known)

Focus on the 80% first. Add the 20% as you grow.

## The test

The agent feels like a butler if:
- A user feels private
- A user feels one step ahead
- A user feels fast
- A user feels known

If any of these fails, the agent feels like a servant.
Fix the agent.

## The lesson

4 principles. 1 butler. 1 lesson: invest in the
principles.

The agent that invests in the principles is the butler.
The agent that doesn't is the servant. The butler is
adopted. The servant is not.

The agent era is here. The butler is the design. The
design is the discipline. The discipline is the value.
