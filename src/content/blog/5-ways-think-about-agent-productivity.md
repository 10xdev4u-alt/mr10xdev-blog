5 ways to think about productivity: speed, accuracy, cost,
scope, time. Each is a different productivity lever. The
framework, the examples, the lesson.

## Lever 1: Speed

The agent is fast. The agent responds in < 1 second. The
agent is productive.

**Examples:**
- "The agent responds in 500ms"
- "The agent streams the first word in 200ms"
- "The agent completes the task in 5 seconds"

**When to use:** the user wants speed. The user is in a
hurry. The agent is interactive.

**Pros:**
- Engaging (the user is engaged)
- Fast (the user doesn't wait)
- Productive (the user gets more done)

**Cons:**
- Cost (speed costs money)
- Quality (speed can hurt quality)
- Risk (speed can cause mistakes)

## Lever 2: Accuracy

The agent is accurate. The agent is right. The agent is
productive.

**Examples:**
- "The agent labels 95% of issues correctly"
- "The agent drafts 90% of release notes correctly"
- "The agent responds to 85% of comments helpfully"

**When to use:** the user wants quality. The user is
patient. The agent is critical.

**Pros:**
- Quality (the user gets quality)
- Trust (the user trusts the agent)
- Productive (the user gets more done)

**Cons:**
- Cost (accuracy costs money)
- Latency (accuracy can take time)
- Complexity (accuracy is hard)

## Lever 3: Cost

The agent is cheap. The agent costs $0.01 per run. The
agent is productive.

**Examples:**
- "The agent costs $0.01 per run"
- "The agent costs $1 per day"
- "The agent costs $30 per month"

**When to use:** the user has a budget. The user is
cost-sensitive. The agent is at scale.

**Pros:**
- Affordable (the user can afford it)
- Scalable (the user can scale)
- Productive (the user gets more done)

**Cons:**
- Quality (cost can hurt quality)
- Speed (cost can hurt speed)
- Scope (cost can limit scope)

## Lever 4: Scope

The agent is broad. The agent handles 10 cases. The
agent is productive.

**Examples:**
- "The agent labels 10 types of issues"
- "The agent handles 5 types of PRs"
- "The agent responds to 8 types of comments"

**When to use:** the user has many use cases. The user
wants one agent. The agent is general.

**Pros:**
- General (the agent is general)
- Comprehensive (the agent is comprehensive)
- Productive (the user gets more done)

**Cons:**
- Quality (scope can hurt quality)
- Cost (scope can hurt cost)
- Complexity (scope is complex)

## Lever 5: Time

The agent saves time. The agent saves 30 minutes per day.
The agent is productive.

**Examples:**
- "The agent saves 30 minutes per day"
- "The agent saves 5 hours per week"
- "The agent saves 20 hours per month"

**When to use:** the user is busy. The user wants to save
time. The agent is delegated.

**Pros:**
- Valuable (the user gets time back)
- Productive (the user is more productive)
- ROI (the user has a clear ROI)

**Cons:**
- Setup (the agent takes time to set up)
- Trust (the user has to trust the agent)
- Risk (the agent can be wrong)

## The 5 together

The 5 are the productivity. The productivity is the value.
The value is the ROI.

| Lever | What it provides | Cost |
|---|---|---|
| Speed | The user is engaged | High |
| Accuracy | The user trusts | High |
| Cost | The user can afford | Low |
| Scope | The user is general | Medium |
| Time | The user gets time | Medium |

The lever that matches the need is the right lever.

## The 80/20

80% of the value comes from:
- Accuracy (the user trusts)
- Time (the user gets time)

20% comes from:
- Speed (the user is engaged)
- Cost (the user can afford)
- Scope (the user is general)

Focus on the 80% first. Add the 20% as you grow.

## The choice

For each agent, ask:
- Is the user in a hurry? (speed)
- Is the user patient? (accuracy)
- Is the user cost-sensitive? (cost)
- Is the user general? (scope)
- Is the user busy? (time)

The right answer is the right lever at the right cost.

## The lesson

5 levers. 1 productivity. 1 lesson: pick the right
combination.

The lever that matches the need is the right lever. The
lever that doesn't match is the wrong lever.

The agent era is here. The productivity is the design.
The design is the choice. The choice is the discipline.
