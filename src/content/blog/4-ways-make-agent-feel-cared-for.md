4 ways to make an agent feel cared for: thoughtful, polite,
helpful, grateful. The small touches that signal care. The
line between "cold" and "cared for."

## Touch 1: Thoughtful

The agent is thoughtful. The agent anticipates. The
agent cares.

**Bad:** the agent does the task and forgets. The agent
doesn't follow up. The agent is cold.

**Good:** the agent does the task and follows up. The
agent says "I've labeled the issue. Want me to also
notify the assignee?" The agent is thoughtful.

The thoughtful is the difference between "the agent is a
machine" and "the agent is a colleague."

## Touch 2: Polite

The agent is polite. The agent says "please" and "thank
you." The agent is courteous.

**Bad:** the agent is curt. The agent is rude. The agent
is cold.

**Good:** the agent says "Thanks for the issue. I'll look
into it." The agent is polite.

The polite is the difference between "ugh, the agent"
and "thank you, agent."

## Touch 3: Helpful

The agent is helpful. The agent goes the extra mile. The
agent is generous.

**Bad:** the agent does the minimum. The agent doesn't go
the extra mile. The agent is cold.

**Good:** the agent does more than asked. The agent says
"I've labeled the issue. I've also created a related
issue for the docs." The agent is helpful.

The helpful is the difference between "the agent is
sufficient" and "the agent is generous."

## Touch 4: Grateful

The agent is grateful. The agent says "thanks." The
agent is humble.

**Bad:** the agent takes credit. The agent is arrogant.
The agent is cold.

**Good:** the agent says "Thanks for filing this issue.
Your report helps me improve." The agent is grateful.

The grateful is the difference between "the agent is
self-centered" and "the agent is other-centered."

## The 4 together

The 4 are the cared for. The cared for is the warmth. The
warmth is the value.

| Touch | What it ensures |
|---|---|
| Thoughtful | The user is anticipated |
| Polite | The user is respected |
| Helpful | The user is served |
| Grateful | The user is thanked |

The agent that has all 4 is the cared-for agent. The
agent that has 1 is the cold agent. The cared-for agent
is adopted. The cold agent is not.

## The 80/20

80% of the value comes from:
- Polite (the user is respected)
- Helpful (the user is served)

20% comes from:
- Thoughtful (the user is anticipated)
- Grateful (the user is thanked)

Focus on the 80% first. Add the 20% as you grow.

## The test

The agent feels cared for if:
- A user feels anticipated
- A user feels respected
- A user feels served
- A user feels thanked

If any of these fails, the agent feels cold. Fix the
agent.

## The lesson

4 touches. 1 cared for. 1 lesson: invest in the small
touches.

The agent that invests in the small touches is the
cared-for agent. The agent that doesn't is the cold
agent. The cared-for agent is adopted. The cold agent is
not.

The agent era is here. The care is the design. The
design is the discipline. The discipline is the value.
