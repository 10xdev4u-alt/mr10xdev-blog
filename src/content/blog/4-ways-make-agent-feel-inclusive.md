4 ways to make an agent feel inclusive: accessible,
multilingual, affordable, welcoming. The small touches that
signal openness. The line between "us" and "everyone."

## Way 1: Accessible

The agent works for everyone. The agent works for users
with disabilities. The agent works with screen readers.
The agent works with keyboard-only navigation.

**Examples:**
- All UI elements are keyboard-accessible
- All text has alt text for images
- All colors meet contrast standards
- All interactions have a fallback

**When to use:** the agent has a UI. The agent is for the
public. The agent is for the world.

**Pros:**
- Inclusive (the agent is for everyone)
- Compliant (the agent meets regulations)
- Discoverable (the agent is findable by search)

**Cons:**
- Effort (the agent requires care)
- Cost (the agent requires testing)
- Slow (the agent requires iteration)

## Way 2: Multilingual

The agent speaks the user's language. The agent supports
many languages. The agent doesn't force English.

**Examples:**
- English, Spanish, French, German, Japanese, Chinese
- Auto-detect the user's language
- Translate the agent's responses
- Translate the docs

**When to use:** the agent is global. The user is diverse.
The agent is for the world.

**Pros:**
- Inclusive (the agent is for everyone)
- Aligned (the agent meets the user where they are)
- Discoverable (the agent is findable in any language)

**Cons:**
- Effort (the agent requires translation)
- Cost (the agent requires LLM calls)
- Quality (the agent's translations may be wrong)

## Way 3: Affordable

The agent is free or cheap. The agent has a free tier. The
agent doesn't lock out the user.

**Examples:**
- Free tier: 100 runs / month
- $5/month: 1000 runs / month
- $50/month: unlimited runs

**When to use:** the agent is for the public. The user has
a budget. The agent is for the world.

**Pros:**
- Inclusive (the agent is for everyone)
- Aligned (the user can try before they buy)
- Discoverable (the user can find the agent)

**Cons:**
- Cost (the free tier costs the vendor)
- Abuse (the free tier can be abused)
- Support (the free tier users need support)

## Way 4: Welcoming

The agent is friendly. The agent greets the user. The
agent makes the user feel at home.

**Examples:**
- "Hi! I'm the triage agent. How can I help?"
- "Welcome back! What can I do for you today?"
- "I see you're new here. Let me show you around."

**When to use:** the agent is for the public. The user is a
stranger. The agent is for the world.

**Pros:**
- Inclusive (the agent is for everyone)
- Aligned (the agent makes the user feel welcome)
- Discoverable (the agent is memorable)

**Cons:**
- Effort (the agent requires care)
- Cost (the agent requires extra LLM calls)
- Risk (the agent may be too friendly)

## The 4 together

The 4 are the inclusive. The inclusive is the openness. The
openness is the value.

| Way | What it ensures | Cost |
|---|---|---|
| Accessible | The agent is for everyone | High |
| Multilingual | The agent is in any language | Medium |
| Affordable | The agent is for any budget | High |
| Welcoming | The agent is for any user | Low |

The agent that has all 4 is the inclusive agent. The agent
that has 1 is the exclusive agent. The inclusive agent is
adopted. The exclusive agent is not.

## The 80/20

80% of the value comes from:
- Affordable (the user can try)
- Welcoming (the user wants to try)

20% comes from:
- Accessible (the user can use)
- Multilingual (the user can read)

Focus on the 80% first. Add the 20% as you grow.

## The test

The agent feels inclusive if:
- A user with a disability can use it
- A user who speaks only Spanish can use it
- A user who has $0 can use it
- A user who is new can use it

If any of these fails, the agent is exclusive. Fix the
agent.

## The lesson

4 ways. 1 inclusive. 1 lesson: invest in the small
touches.

The agent that invests in the small touches is the
inclusive agent. The agent that doesn't is the exclusive
agent. The inclusive agent is adopted. The exclusive
agent is not.

The agent era is here. The inclusive is the design. The
design is the discipline. The discipline is the openness.
