4 ways to make an agent feel inevitable: necessary, obvious,
easy, natural. The principles that make agents feel like
they should exist. The line between "novel" and
"inevitable."

## Principle 1: Necessary

The agent is necessary. The user needs the agent. The
user can't live without the agent.

**Bad:** the agent is a nice-to-have. The user can live
without the agent. The agent is a toy.

**Good:** the agent is a must-have. The user can't live
without the agent. The agent is essential.

The necessary is the difference between "the agent is a
toy" and "the agent is inevitable."

## Principle 2: Obvious

The agent is obvious. The user knows what the agent
does. The user doesn't have to think.

**Bad:** the agent is mysterious. The user has to read
the docs. The user is confused.

**Good:** the agent is obvious. The user knows what the
agent does. The user doesn't have to think.

The obvious is the difference between "the agent is a
mystery" and "the agent is inevitable."

## Principle 3: Easy

The agent is easy. The user can use the agent in 1
minute. The user doesn't have to learn.

**Bad:** the agent is hard. The user has to read the
docs. The user has to learn. The user gives up.

**Good:** the agent is easy. The user can use the agent
in 1 minute. The user is hooked.

The easy is the difference between "the agent is work"
and "the agent is inevitable."

## Principle 4: Natural

The agent is natural. The agent feels like a natural
extension of the user's workflow. The agent is part of
the user's life.

**Bad:** the agent is artificial. The user has to change
their workflow. The user is disrupted.

**Good:** the agent is natural. The agent fits the
user's workflow. The user is delighted.

The natural is the difference between "the agent is
disruptive" and "the agent is inevitable."

## The 4 together

The 4 are the inevitable. The inevitable is the future.
The future is the value.

| Principle | What it ensures |
|---|---|
| Necessary | The user needs it |
| Obvious | The user knows it |
| Easy | The user uses it |
| Natural | The user loves it |

The agent that has all 4 is the inevitable agent. The
agent that has 1 is the novel agent. The inevitable agent
is adopted. The novel agent is not.

## The 80/20

80% of the value comes from:
- Necessary (the user needs it)
- Easy (the user uses it)

20% comes from:
- Obvious (the user knows it)
- Natural (the user loves it)

Focus on the 80% first. Add the 20% as you grow.

## The test

The agent feels inevitable if:
- A user needs it
- A user knows it
- A user uses it
- A user loves it

If any of these fails, the agent feels novel. Fix the
agent.

## The lesson

4 principles. 1 inevitable. 1 lesson: invest in the
principles.

The agent that invests in the principles is the
inevitable agent. The agent that doesn't is the novel
agent. The inevitable agent is adopted. The novel agent
is not.

The agent era is here. The inevitable is the design. The
design is the discipline. The discipline is the value.
